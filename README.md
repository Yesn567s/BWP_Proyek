# BWP_Proyek
Ini adalah proyek kami mengenai BWP(Backend Web Programming), program yang kami buat adalah Tixly yang memiliki banyak fitur 

di sini terdapat 2 role yaitu admin dan user 

admin dapat mengatur berbagai hal untuk keperluan produksi seperti movie baru, promo voucher ataupun ticket dan event

user dapat berbelanja di web dan dapat menemukan banyak deal atau informasi yang sangat berguna untuk user
