# BWP_Proyek
Tixxly adalah sebuah website ticketing yang memungkinkan pengguna untuk membeli berbagai macam tiket secara online dengan mudah dan cepat. Tixxly menyediakan beragam kategori tiket, mulai dari tiket cinema, tiket museum, tiket hiburan, hingga berbagai event lainnya. Selain pembelian tiket, Tixxly juga menghadirkan konten pendukung seperti blog dan food menu untuk meningkatkan pengalaman pengguna.

Website ini dilengkapi dengan Admin Page yang komprehensif, sehingga admin dapat mengelola konten, transaksi, serta memantau performa bisnis melalui dashboard statistik.

Program kami memiliki 2 role yaitu admin dan user, dengan gmail admin@gmail.com dan pass:123, serta user joni@mail.com dengan pass:12345
